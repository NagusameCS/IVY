# 8.7 Definition of a Group

## Objectives
- Group axioms
- Latin square property of operation table
- Abelian groups

## 1. Group
- Set G with * satisfying closure, associativity, identity, inverses

## 2. Latin Square
- Each element appears once per row/column in Cayley table

## 3. Abelian
- Commutative group
