# 4.2 Scalar (Dot) Product

## Objectives
- Define scalar product
- Use properties
- Determine angle & perpendicularity / parallelism

## 1. Definition
- v·w = v1 w1 + v2 w2 + v3 w3
- Also = |v||w| cos θ

## 2. Properties
- Commutative: v·w = w·v
- Distributive
- (k v)·w = k(v·w)
- v·v = |v|^2

## 3. Angle
- cos θ = (v·w)/(|v||w|)
- Perpendicular if v·w=0
- Parallel if v = k w
