Title: Definition of a Group
Curriculum: 8.8.7 - Sets, Relations & Groups (HL Option)
Subject: math_aa

Begin by explaining what a group is and how it works in mathematics.

## Key Concepts (main learning points)

1. **Definition:** Grouping objects with the same properties or characteristics.
2. **Relationship:** A set of objects that have an equal number of elements, meaning they can be arranged into pairs or families without any overlap.
3. **Examples:** The set {a, b}, {b, c}, and {c, d} form a group since their elements are unique to each other (set-like) and we can arrange them in pairs.
4. **Properties:** For example, the set of all even numbers is an example of a group because it has three properties: closure (all even numbers have the property that they're not divisible by 2), associativity (the order in which two elements are combined does not affect the result) and the existence of inverse operations.
5. **Examples:** The set of integers, the set of natural numbers, the set of divisors... Each group has a unique property that distinguishes it from other groups like the circle or the rectangle.
