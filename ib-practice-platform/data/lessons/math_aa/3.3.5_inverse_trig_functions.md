## Inverse Trig Functions

## Overview

This lesson focuses on the concept of **inverse trig functions**, exploring their graphs, properties, and applications within the context of right-angled triangles and trigonometric equations. Students will learn to find the principal value of inverse trig functions and apply them to solve for angles and side lengths.

## Key Concepts

*   Understanding the relationship between trigonometric functions and their inverse functions.
*   Graphing inverse trig functions and identifying key features.
*   Determining the principal value of inverse trig functions.
*   Solving trigonometric equations involving inverse trig functions.
*   Applying inverse trig functions to solve problems involving right-angled triangles.

### 1. Introduction to Inverse Functions

> **Note:** An inverse function reverses the input and output of a given function. If *f(x)* = *y*, then *f^{-1}(y)* = *x*.

Inverse functions are fundamental in mathematics. They allow us to 'undo' operations performed by trigonometric functions, enabling us to solve for angles when we know the ratio of sides in a right-angled triangle. We'll start by reviewing the standard trigonometric functions: sine, cosine, and tangent.

### 2. Definitions of Inverse Trig Functions

The **inverse trig functions** are defined as follows:

*   **arcsin(x)** (or sin^{-1}(x)):  Returns the angle whose sine is *x*.  The range is [-π/2, π/2] (or -90° to 90°).
*   **arccos(x)** (or cos^{-1}(x)): Returns the angle whose cosine is *x*.  The range is [0, π] (or 0° to 180°).
*   **arctan(x)** (or tan^{-1}(x)): Returns the angle whose tangent is *x*. The range is (-π/2, π/2) (or -90° to 90°).

> **Important:** The notation sin^{-1}(x) is *not* the same as 1/sin(x). It represents the inverse function.

### 3. Graphs of Inverse Trig Functions

Let’s examine the graphs of the inverse trig functions.  These graphs are periodic and resemble the graphs of the original trig functions, but with their ranges reflected.

*   **arcsin(x):**  Graph of y = arcsin(x) has a range of [-π/2, π/2].  The graph is symmetric about the y-axis.
*   **arccos(x):** Graph of y = arccos(x) has a range of [0, π].  The graph is symmetric about the point (π/2, 0).
*   **arctan(x):** Graph of y = arctan(x) has a range of (-π/2, π/2). The graph is symmetric about the origin.

> **Exam Tip:**  Recognizing the key features of inverse trig function graphs (periodicity, symmetry, range) is crucial for solving problems.

**Example 1:**

Given the graph of arcsin(x), sketch the graph of y = arcsin(2x).  Consider the scaling of both the x and y axes.

*   **Solution:** The graph of y = arcsin(2x) is a horizontally scaled version of y = arcsin(x) by a factor of 2. The range of y = arcsin(2x) is [-π/3, π/3], which corresponds to the shrinking of the original range.  The key points would be (-π/6, -π/3), (-π/3, -π/3), (-π/6, π/3), etc.

### 4. Principal Values

The **principal value** of an inverse trig function is the unique angle within its range that satisfies the given input.  For example, arcsin(0.5) has a principal value of π/6 (30°), because π/6 is the only angle between -π/2 and π/2 whose sine is 0.5.

> **Example:**
> - arcsin(1) = π/2 (90°)
> - arccos(1) = 0 (0°)
> - arctan(1) = π/4 (45°)

### 5. Solving Trigonometric Equations

Inverse trig functions are invaluable for solving trigonometric equations.

**Example 2:**

Solve the equation sin(θ) = 1/2, where 0 ≤ θ ≤ π.

*   **Solution:**
    1.  Take the inverse sine of both sides: θ = arcsin(1/2)
    2.  The principal value of arcsin(1/2) is π/6.
    3.  Therefore, θ = π/6 is one solution.  Since sin(θ) is positive in the first and second quadrants, the other solution is: θ = π - π/6 = 5π/6.

### 6. Application: Right-Angled Triangles

Inverse trig functions can be used to find angles in right-angled triangles when we know the ratios of the sides (SOH CAH TOA).

**Example 3:**

In a right-angled triangle ABC, angle B = 25° and AB = 10 cm. Find the length of AC.

*   **Solution:**
    1.  We have tan(B) = opposite/adjacent = AC/AB.
    2.  Therefore, tan(25°) = AC/10.
    3.  AC = 10 * tan(25°)
    4.  Using a calculator, tan(25°) ≈ 0.4663
    5.  AC ≈ 10 * 0.4663 = 4.663 cm
