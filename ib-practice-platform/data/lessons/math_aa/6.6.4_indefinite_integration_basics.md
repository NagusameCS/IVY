Title: Indefinite Integration Basics

Introduction
The goal of this lesson is to provide an overview of indefinite integration techniques that will be applied throughout the course to help students develop a deep understanding and confidence in their mathematical abilities.

Prerequisites
- Students will need to have some basic knowledge of calculus prior to starting the course.
- They should understand that definite integrals are only defined for a specific range of values, whereas indefinite integrals treat functions as points on a curve rather than an entire region.

**Introduction (3-5 minutes)**
The instructor would typically start the introduction by presenting some key concepts and definitions related to definite and indefinite integrals. This could include:

* Fundamental theorem of calculus (which is crucial in differentiation)
* Differentiability (definition for functions, which is necessary for integration steps like integrating a function or determining limits)
* The Fundamental Theorem of Calculus (a cornerstone of calculus).

**Definite Integrals**
We are about to dive into the more advanced topic of definite integrals. A definite integral of a function f(x) over an interval [a, b] is defined as:

∫[a, b] f(x) dx = F(b) - F(a)

This definition will be applied to the given problem we are currently discussing (indefinite integration basics). In this case, we are finding the antiderivative of the function 1/(2x + a), which is not defined at x = a. This highlights the importance of applying definite integrals correctly in calculus problems such as these.

**Example (5-8 minutes)**
Let's consider an example where you want to find an antiderivative for a given function f(x). One commonly used method involves integrating both sides, but it can be tedious and not always clear how the result is obtained from those two parts of integration. This problem helps students understand that integration requires evaluating definite integrals.

**Example (10-20 minutes)**
Consider an example where we want to find the antiderivative of f(x) = 3 / x^2 + 4i, which is also not defined at x = -4 because it does not match our desired behavior for this function on the interval [-4, 4]. This problem demonstrates how integration can be used correctly even when there are no definite integrals in sight.
