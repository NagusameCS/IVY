## Key Concepts

This lesson will explore the **vector equation of a line** within the IB Mathematics AA framework (4.4.3). We will learn to represent lines geometrically using vector equations, understand the relationship between the vector equation and parametric equations, and apply these concepts to solve problems. *Mastering this concept is crucial for understanding 3D geometry and transformations.*

## 1. Introduction to Vector Equations of Lines

> **Important:** The vector equation of a line provides a powerful way to define a line in 3D space, relating any point on the line to a specific point (the **point of projection**) and a **direction vector**.

The vector equation of a line can be written as:

$\vec{r} = \vec{a} + t\vec{d}$

Where:
*   $\vec{r}$ is the position vector of any point on the line.
*   $\vec{a}$ is the **point of projection** – a fixed point on the line.
*   $\vec{d}$ is the **direction vector** – a vector parallel to the line.
*   *t* is a **scalar parameter** – any real number that determines the position of the point on the line.

Let's examine this more closely. The parameter *t* essentially allows us to ‘move’ along the line from the point of projection $\vec{a}$ in the direction of $\vec{d}$.

### 1.1 Point of Projection and Direction Vector

The **point of projection** ($\vec{a}$) is a key component. It provides a fixed reference point on the line.  The **direction vector** ($\vec{d}$) dictates the line’s orientation.  A non-zero direction vector is crucial; a zero vector would represent a point, not a line.

> **Example:** Consider a line passing through the point A(1, 2, 3) in 3D space.  We can set $\vec{a} = \langle 1, 2, 3 \rangle$.

## 2. Finding the Vector Equation from a Point and Direction

Given a point on a line and a direction vector, we can determine the vector equation.

### 2.1  From Two Points

If we are given two points, $P_1(x_1, y_1, z_1)$ and $P_2(x_2, y_2, z_2)$ on the line, we can calculate the direction vector as:

$\vec{d} = \vec{P_2} - \vec{P_1} = \langle x_2 - x_1, y_2 - y_1, z_2 - z_1 \rangle$

Then, we can choose either point as the point of projection,  $\vec{a} = \vec{P_1}$.  The vector equation is then:

$\vec{r} = \vec{P_1} + t\vec{d}$

### 2.2  Worked Example 1

**Problem:** Find the vector equation of the line passing through the point P(2, -1, 5) in the direction of the vector $\vec{d} = \langle 3, -2, 1 \rangle$.

**Solution:**

1.  Identify the point of projection: $\vec{a} = \langle 2, -1, 5 \rangle$.
2.  Use the given direction vector: $\vec{d} = \langle 3, -2, 1 \rangle$.
3.  Apply the vector equation: $\vec{r} = \vec{a} + t\vec{d} = \langle 2, -1, 5 \rangle + t\langle 3, -2, 1 \rangle$

Therefore, the vector equation of the line is:
$\vec{r} = \langle 2 + 3t, -1 - 2t, 5 + t \rangle$

## 3. Parametric Equations of a Line

> **Note:** The vector equation and parametric equations are equivalent representations of the same line.

The vector equation can be easily converted into parametric equations.  Since *t* is a parameter, we can express the coordinates of any point on the line as follows:

$x = 2 + 3t$
$y = -1 - 2t$
$z = 5 + t$

### 3.1  Converting between Vector and Parametric Equations

The conversion is straightforward:

*   $x = a_x + td_x$
*   $y = a_y + td_y$
*   $z = a_z + td_z$

Where $(a_x, a_y, a_z)$ is the point of projection and $\vec{d} = \langle d_x, d_y, d_z \rangle$ is the direction vector.

## 4. Applications of Vector Equations

> **Exam Tip:**  Practice converting between vector and parametric equations. This is a common exam technique.

Vector equations are particularly useful in solving problems involving lines in 3D space, such as finding the shortest distance between two lines, or determining if two lines are parallel or intersecting.

### 4.1  Finding the Distance Between Two Points

The distance between two points $P_1$ and $P_2$ can be found using the vector equation.

Let $P_1 = \vec{a}$ and $P_2 = \vec{a} + \vec{d}$ where $\vec{d}$ is the direction vector. Then the distance $d$ between $P_1$ and $P_2$ is given by:

$d = ||\vec{d}||$

## 5. Worked Examples

### 5.1 Worked Example 2

**Problem:** Find the vector equation of the line passing through the point B(-1, 4, 2) and parallel to the vector $\vec{v} = \langle 2, 0, -1 \rangle$.  Also, find the parametric equations.

**Solution:**

1.  Point of projection: $\vec{a} = \langle -1, 4, 2 \rangle$.
2.  Direction vector: $\vec{v} = \langle 2, 0, -1 \rangle$.
3.  Vector equation: $\vec{r} = \vec{a} + t\vec{v} = \langle -1, 4, 2 \rangle + t\langle 2, 0, -1 \rangle$.
    This gives us $\vec{r} = \langle -1 + 2t, 4, 2 - t \rangle$.
4.  Parametric equations:
    $x = -1 + 2t$
    $y = 4$
    $z = 2 - t$

### 5.2 Worked Example 3

**Problem:**  Given the vector equation $\vec{r} = \langle 1, -2, 3 \rangle + t\langle 4, 5, -2 \rangle$, find the coordinates of a point on the line where *t* = 2.

**Solution:**

1.  Substitute *t* = 2 into the vector equation:
    $\vec{r} = \langle 1, -2, 3 \rangle + 2\langle 4, 5, -2 \rangle$
2.  Simplify:
    $\vec{r} = \langle 1 + 8, -2 + 10, 3 - 4 \rangle = \langle 9, 8, -1 \rangle$
3.  Therefore, the coordinates of the point are (9, 8, -1).

## 6. Summary

The vector equation of a line, $\vec{r} = \vec{a} + t\vec{d}$, is a fundamental tool for representing lines in 3D space.  It uses a **point of projection** ($\vec{a}$) and a **direction vector** ($\vec{d}$) to define the line. *This approach allows us to easily generate parametric equations* for the line, providing an alternative way to describe its position.  Understanding the relationship between the vector and parametric equations is crucial for problem-solving in 3D geometry.  The ability to derive the vector equation from given information and convert between representations is a key skill assessed in the IB Math AA syllabus.
