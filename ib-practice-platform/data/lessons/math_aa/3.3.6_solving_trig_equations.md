## Solving Trig Equations

## Overview

This lesson focuses on solving trigonometric equations, utilizing both algebraic manipulation and the understanding of trigonometric identities. Students will learn to solve equations involving sine, cosine, and tangent, applying their knowledge of unit circles and trigonometric graphs.

## Key Concepts

*   Solving trigonometric equations requires applying trigonometric identities to isolate the trigonometric functions.
*   Understanding the unit circle and its relationship to trigonometric functions is crucial for determining solutions within specific intervals.
*   Equations can have multiple solutions, and students must consider the period of the trigonometric functions when finding all valid solutions.
*   *Exam Tip:* Practice solving equations over various intervals to fully understand the concept.

### 1. Understanding the Unit Circle and Trig Functions

> **Important:** The unit circle provides a visual representation of trigonometric functions and their relationships. The coordinates of a point on the unit circle corresponding to an angle θ are defined as (cos θ, sin θ).

The **unit circle** is a circle with a radius of 1 centered at the origin of the Cartesian coordinate system. This circle is fundamental to understanding trigonometric functions.

*   **Sine (sin θ):**  The y-coordinate of a point on the unit circle.  `sin θ = y/r`
*   **Cosine (cos θ):** The x-coordinate of a point on the unit circle. `cos θ = x/r`
*   **Tangent (tan θ):** The ratio of sine to cosine: `tan θ = sin θ / cos θ`

```python
import math

def sine(angle_degrees):
  """Calculates the sine of an angle in degrees."""
  angle_radians = math.radians(angle_degrees)
  return math.sin(angle_radians)

def cosine(angle_degrees):
  """Calculates the cosine of an angle in degrees."""
  angle_radians = math.radians(angle_degrees)
  return math.cos(angle_radians)

def tangent(angle_degrees):
  """Calculates the tangent of an angle in degrees."""
  angle_radians = math.radians(angle_degrees)
  return math.tan(angle_radians)
```

**Example:**  If θ = 30°, then cos(30°) = √3/2 and sin(30°) = 1/2.

### 2. Solving Simple Trigonometric Equations

Let's start with some fundamental equations.

**Example 1:**

Solve:  sin(x) = 1  for 0 ≤ x < 2π

1.  Recognize that sin(x) = 1 when x = π/2.
2.  Since sine is a periodic function with a period of 2π, we add 2πk to our solution, where k is an integer:  x = π/2 + 2πk

Therefore, the solutions are x = π/2, π/2 + 2π, π/2 + 4π, etc.
For the interval 0 ≤ x < 2π, the solution is x = π/2.

**Example 2:**

Solve: cos(x) = 0 for 0 ≤ x < 2π

1.  Cosine is equal to 0 when x = π/2 and x = 3π/2.
2.  These are the only solutions within the given interval.

**Example 3:**

Solve: tan(x) = 1 for 0 ≤ x < 2π

1.  Tangent is equal to 1 when x = π/4 and x = 5π/4.
2.  These are the only solutions within the given interval.

### 3. Using Trigonometric Identities

Trigonometric identities are crucial for solving equations. Here are some key ones:

*   sin²(x) + cos²(x) = 1
*   1 + tan²(x) = sec²(x)
*   1 + cot²(x) = csc²(x)

**Example:**  Solve: 2cos²(x) - 1 = 0

1.  Start with the equation: 2cos²(x) - 1 = 0
2.  Rearrange: 2cos²(x) = 1
3.  cos²(x) = 1/2
4.  cos(x) = ±√(1/2) = ±√2/2
5.  Therefore, x = π/4, 5π/4, 7π/4, 9π/4  (within 0 ≤ x < 2π)

### 4. Solving Equations with Multiple Solutions

Many trigonometric equations have infinitely many solutions. It’s important to consider the **period** of the trigonometric function to determine all solutions within a given interval.

**Example:** Solve sin(3x) = sin(x) for 0 ≤ x < 2π

1.  Since sin(3x) = sin(x), we have 3x = x + 2πk or 3x = π - x + 2πk, where k is an integer.
2.  Case 1: 3x = x + 2πk  => 2x = 2πk => x = πk
    For 0 ≤ x < 2π, we have x = 0, π, 2π.
3.  Case 2: 3x = π - x + 2πk  => 4x = π + 2πk => x = (π/4) + (πk/2)
     For 0 ≤ x < 2π, we have x = (π/4), (5π/4), (9π/4), (13π/4)

Therefore, the solutions are 0, π, 2π, π/4, 5π/4, 9π/4, 13π/4.

### 5.  Solving Equations over Specific Intervals

Often, exam questions will specify an interval to solve within.  This means you'll only need to find the solutions that fall within that range.

**Example:** Solve sin(x) = 0.5 for 0 ≤ x ≤ 2π

1.  We know that sin(x) = 0.5 when x = π/6 and x = 5π/6.
2.  Both of these solutions lie within the interval [0, 2π].

### 6.  Advanced Techniques (HL Extension)

*   **Reduction Formulas:**  These are used to simplify trigonometric expressions and solve equations involving angles greater than 2π.
*   **Complex Solutions:**  Trigonometric equations can sometimes have solutions involving complex numbers, requiring the use of Euler's formula (e^{ix} = cos(x) + i sin(x)). *Note:* This is typically introduced in HL.
