## Polynomial Functions

## Overview

This lesson explores polynomial functions, their properties, and how to analyze them. We’ll cover different types of polynomials, their graphs, and key characteristics like roots and turning points. This builds a strong foundation for later topics within the Functions strand.

## Key Concepts

*   Understanding the general form of polynomial functions.
*   Identifying and classifying polynomial functions by degree and number of terms.
*   Analyzing the relationship between polynomial coefficients and the roots.
*   Graphing polynomial functions and interpreting their features.
*   Applying polynomial functions to model real-world situations.

### 1. Introduction to Polynomial Functions

> **Important:** A **polynomial function** is a function that can be expressed in the form:

$$f(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0$$

where *n* is a non-negative integer and the coefficients *a~n~, a~n-1~, ..., a~1~, a~0~* are constants.

*   The **degree** of the polynomial is the highest power of *x* (i.e., *n*).
*   Polynomial functions are fundamental in many areas of mathematics, science, and engineering.

**Example:**
$$f(x) = 3x^4 - 2x^2 + 7x - 1$$
Here, *n* = 4, and the degree is 4. The coefficients are 3, -2, 7, and -1.

**Worked Example 1:**
Identify the degree and leading coefficient of the following polynomial:
$$g(x) = -5x^3 + 2x - 8$$
**Solution:** The highest power of *x* is 3, so the degree is 3. The leading coefficient is -5.

**Worked Example 2:**
Determine if the following is a polynomial:
$$h(x) = x^{7} + 4x^2 - 10$$
**Solution:** This is a polynomial because all exponents of *x* are non-negative integers. The degree is 7, and the leading coefficient is 1.

### 2. Types of Polynomials

Polynomials can be classified based on their degree and the number of terms:

*   **Linear Polynomial:** Degree 1, one term (e.g., *f(x) = 2x + 1*)
*   **Quadratic Polynomial:** Degree 2, one term (e.g., *f(x) = x^{2}*)
*   **Cubic Polynomial:** Degree 3, one term (e.g., *f(x) = 2x^{3} - 3*)
*   **Quartic Polynomial:** Degree 4, one term (e.g., *f(x) = x^{4} + 5x^{2} - 7*)
*   **Multi-term Polynomial:** Has more than one term. *Example:* *f(x) = 2x^{3} + x^{2} - 5x + 3*

**Exam Tip:**  Understanding the terminology surrounding polynomial degrees and terms is crucial for IB exam questions.

### 3. Roots and Zeros of Polynomial Functions

*   The **zero** (or **root**) of a polynomial function *f(x)* is a value of *x* for which *f(x) = 0*. These values are the *x*-intercepts of the graph of the function.
*   A polynomial of degree *n* can have at most *n* real roots.

**Example:**
For the polynomial *f(x) = x^{2} - 4*, the roots are *x = 2* and *x = -2*, because *f(2) = 0* and *f(-2) = 0*.

**Worked Example 3:**
Find the roots of the polynomial:  $$p(x) = x^3 - 6x^2 + 11x - 6$$
**Solution:** We can test integer values for *x* as potential roots.  Trying *x = 1*, we get *p(1) = 1 - 6 + 11 - 6 = 0*.  Therefore, *x = 1* is a root. We can then divide *x^{3} - 6x^{2} + 11x - 6* by *(x - 1)* using polynomial long division or synthetic division. This results in *(x - 2)(x - 3)* as the other factors.  Therefore, the roots are *x = 1, 2, and 3*.

### 4. Analyzing Polynomial Graphs

*   The graph of a polynomial function typically shows a smooth curve.
*   For polynomials of degree 1 or 2, the graphs are lines or parabolas, respectively.
*   For polynomials of degree 3 or higher, the graphs can have more complex shapes, including turning points (local maxima and minima).

**Note:** The sign of the leading coefficient ( *a~n~* ) determines the end behavior of the graph. If *a~n~* > 0, the graph goes to +∞ as *x* goes to +∞ and -∞ as *x* goes to -∞. If *a~n~* < 0, the graph goes to -∞ as *x* goes to +∞ and +∞ as *x* goes to -∞.

### 5. Turning Points and Relative Extrema

> **Important:** Turning points are points on the graph of a polynomial function where the function changes its direction (from increasing to decreasing or vice-versa).  Relative extrema are the *x*-values of these turning points.

*   The number of turning points a polynomial function can have is at most (degree of polynomial - 1).
*   The *x*-values of the turning points can be found by setting the first derivative of the polynomial function equal to zero and solving for *x*.

**Example:**
Consider the quadratic function *f(x) = x^{2} - 4x + 3*.
*   The derivative is *f'(x) = 2x - 4*.
*   Setting *f'(x) = 0*, we get *2x - 4 = 0*, which gives *x = 2*.
*   The turning point is at *x = 2*.  The *y*-value of the turning point is *f(2) = 2^{2} - 4(2) + 3 = -1*.  So, the turning point is (2, -1).

### 6. Modeling with Polynomial Functions

> **Example:** A company's profit *P* (in thousands of dollars) is given by the function *P(x) = -0.01x^{2} + 0.2x + 50*, where *x* is the number of units produced.  Using this function, predict the profit when 100 units are produced.
**Solution:**  Substitute *x = 100* into the function:
*P(100) = -0.01(100)^{2} + 0.2(100) + 50 = -0.01(10000) + 20 + 50 = -100 + 20 + 50 = -30*
Therefore, the profit is -30 thousand dollars. (This example illustrates that polynomial functions can be used to model real-world scenarios, although the negative profit highlights potential limitations).
