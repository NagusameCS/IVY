## Basic Function Families

## Overview

This lesson focuses on understanding the core characteristics of linear, quadratic, and cubic function families. Students will explore their graphs, equations, and key properties, preparing them for more complex function analysis in later IB Math AA modules.

## Key Concepts

*   Students will be able to identify and graph linear, quadratic, and cubic functions.
*   Students will understand the impact of coefficient values on the shape and position of function graphs.
*   Students will be able to analyze and compare the key characteristics of these function families (e.g., rate of change, turning points).
*   Students will develop problem-solving skills by manipulating function equations and interpreting their results.

### Linear Functions (A.1.1)

> **Important:** Linear functions are defined by the equation *y = mx + c*, where 'm' represents the slope and 'c' represents the y-intercept.

1.  **Graphing Linear Functions:**  We begin with the simplest function family: the linear function.  The graph of a linear function is a straight line.

    *   **Example 1:**  Given the equation *y = 2x + 1*, sketch the graph.
        *   To sketch the graph, we can find the y-intercept (c = 1) and use the slope (m = 2) to find another point.  From the y-intercept (0, 1), we rise 2 units and run 1 unit to the right, marking the point (1, 3).  Then, draw a straight line through these two points.
    *   **Example 2:**  Given the equation *y = -3x + 5*, sketch the graph.
        *   The y-intercept is (0, 5) and the slope is -3.  From the y-intercept, we fall 3 units and run 1 unit to the right to the point (1, 2). Draw the line.
    *   **Example 3:**  Given the equation *y = 0.5x - 2*, sketch the graph.
        *   The y-intercept is -2 and the slope is 0.5. From the y-intercept, we rise 0.5 units and run 1 unit to the right to the point (1, -1.5).  Draw the line.
2.  **Slope and Intercepts:**  The slope dictates the steepness and direction of the line, while the y-intercept determines the point where the line crosses the y-axis.  *Slope = rise / run* and *y-intercept = b* in the equation *y = mx + b*.

### Quadratic Functions (A.1.2)

> **Note:** Quadratic functions are defined by the equation *y = ax² + bx + c*, where *a*, *b*, and *c* are constants and *a ≠ 0*.

1.  **Parabola Shape:**  The graph of a quadratic function is a parabola. The coefficient *a* determines whether the parabola opens upwards (a > 0) or downwards (a < 0).
    *   **Example 1:**  Given the equation *y = x²*, sketch the graph.  Here, a = 1. The parabola opens upwards.
    *   **Example 2:**  Given the equation *y = -2x² + 3*, sketch the graph.  Here, a = -2, which means the parabola opens downwards.
    *   **Example 3:**  Given the equation *y = (1/2)x² - 4*, sketch the graph. Here, a = 1/2, which means the parabola opens upwards.

2.  **Vertex:** The vertex of a parabola is the turning point. The x-coordinate of the vertex is given by *x = -b / 2a*.  The y-coordinate is found by substituting this x-value back into the equation.
    *   **Exam Tip:**  Knowing the vertex formula is crucial for sketching and analyzing quadratic functions.

### Cubic Functions (A.1.3)

> **Important:** Cubic functions are defined by the equation *y = ax³ + bx² + cx + d*, where *a*, *b*, *c*, and *d* are constants and *a ≠ 0*.

1.  **Graphing Cubic Functions:** The graph of a cubic function is a curve. The rate of change can vary, and cubic functions can have multiple turning points.
    *   **Example 1:** Given the equation *y = x³*, sketch the graph. The graph increases rapidly.
    *   **Example 2:** Given the equation *y = 2x³ - 3*, sketch the graph. The graph will have a turning point.
    *   **Example 3:** Given the equation *y = (x - 1)(x + 1)(x - 2)*, expand it to find the equation in standard form and then sketch the graph. This will reveal key features like roots and turning points.  Expanding gives *y = (x² - 1)(x - 2) = x³ - 2x² - x + 2*

2.  **Roots (Zeros):**  The roots of a cubic function are the x-values where the graph crosses the x-axis (y = 0).  Finding roots involves solving the cubic equation. This often involves numerical methods or factoring, which may be beyond the scope of introductory IB Math AA.
