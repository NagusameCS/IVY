## First-Order Differential Equations

## Overview

This lesson covers **first-order differential equations** (DEs), methods of solving them, and their applications. Students will learn separation of variables, integrating factors, and how to set up DEs from real-world problems.

## Key Concepts

- Separation of variables
- Integrating factor method
- Modelling with differential equations
- Slope fields

### Separation of Variables

> **Important:** A separable DE can be written as $\frac{dy}{dx} = f(x) \cdot g(y)$, allowing us to separate variables and integrate both sides.

$$\int \frac{1}{g(y)} \, dy = \int f(x) \, dx$$

**Example 1:** Solve $\frac{dy}{dx} = 2xy$ with $y(0) = 3$.

Separating: $\frac{1}{y} dy = 2x \, dx$

Integrating: $\ln|y| = x^2 + C$

So $y = Ae^{x^2}$. Using $y(0) = 3$: $A = 3$, giving $y = 3e^{x^2}$.

**Example 2:** Solve $\frac{dy}{dx} = \frac{x}{y}$

$$y \, dy = x \, dx \implies \frac{y^2}{2} = \frac{x^2}{2} + C \implies y^2 - x^2 = k$$

### Integrating Factor Method

For a linear first-order DE of the form $\frac{dy}{dx} + P(x)y = Q(x)$:

1. Find the integrating factor: $\mu(x) = e^{\int P(x) \, dx}$
2. Multiply both sides by $\mu(x)$
3. Recognise the left side as $\frac{d}{dx}[\mu(x) \cdot y]$
4. Integrate both sides

**Example 3:** Solve $\frac{dy}{dx} + \frac{y}{x} = x^2$

Here $P(x) = \frac{1}{x}$, so $\mu = e^{\int 1/x \, dx} = e^{\ln x} = x$.

Multiplying: $x\frac{dy}{dx} + y = x^3 \implies \frac{d}{dx}(xy) = x^3$

Integrating: $xy = \frac{x^4}{4} + C \implies y = \frac{x^3}{4} + \frac{C}{x}$

### Applications

Differential equations model many real-world situations:
- **Population growth**: $\frac{dP}{dt} = kP$ gives $P = P_0 e^{kt}$
- **Newton's cooling**: $\frac{dT}{dt} = -k(T - T_s)$
- **Radioactive decay**: $\frac{dN}{dt} = -\lambda N$

## Summary

- Separable DEs are solved by moving variables to opposite sides and integrating
- Linear first-order DEs use the integrating factor method
- DEs model exponential growth/decay, cooling, and many other phenomena
