Title: CDFs, Geometric, Neg Binomial, PGFs

### Content

1. Introduction: This lesson will introduce you to some fundamental concepts of probability theory, focusing on conditional probabilities and geometric distribution.
2. Introduction to Negative Binomial Distribution (NBD): An extension of the binomial distribution that models the number of successes in a fixed number of trials with different proportions. In this lesson, we'll explore the NBD and its applications.
3. Geometric Distribution (G) - The Geometric Distribution is a discrete probability distribution where each trial has an equal chance of occurring. In this lesson, we'll calculate the expected value of the geometric distribution and understand its properties.
4. Negative Binomial Distribution (NBB): A special case of the G that models the number of successes in n independent trials with proportions p1/n0, ..., where each trial has an equal chance of occurring. This lesson will introduce you to the NBB by exploring its properties and applications.
5. Geometric Regression & Mixtures: In this lesson, we'll use the geometric distribution to model relationships between variables. We'll also discuss how to create positive linear regression models using the NBD.
6. Using the NDF (Negative Binomial Distribution in Excel) - The NDF can be used as an interactive tool for modeling negative binomial distributions or creating mixed binary logistic regression models with confidence intervals based on the distribution of the independent variables. We'll apply this concept to a real-world problem and illustrate its applications.
7. Multinomial Regression: A supervised learning method that models categorical outcome data, such as binary labels or continuous responses. We'll use this technique in conjunction with the NDF model to create confidence intervals for different categories based on the distribution of independent variables.
8. Confidence Intervals and Probability Plots: In this lesson, we will analyze the results from using the NBD model to estimate the probability of an outcome being within a certain range using a regression curve or a confidence interval plot. This hands-on approach helps you understand how confidence intervals are constructed for both normal and binomial distributions.
9. Probability Primes: In this lesson, we'll explore the concept of independent trials with different proportions to model the probability of an event occurring. We'll use the NBD distribution as a tool to estimate the expected number of events without knowing their individual probabilities.
10. Using the NBB and Confidence Intervals in Excel: This lesson will show you how to apply the NDF model to create confidence intervals for variables using a suitable spreadsheet program, such as Microsoft Excel.
