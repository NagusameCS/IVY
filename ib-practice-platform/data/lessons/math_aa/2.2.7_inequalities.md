## Key Concepts

This lesson will explore **inequalities** within the context of **functions**, specifically relating to the concept of **domain** and **range**. We will investigate how the solution set of an inequality defines the possible values of a function’s input or output.  This understanding is critical for exam success in the functions topic (A.2.7).

## Key Concepts

### 1. Introduction to Inequalities

> **Important:** An **inequality** expresses a relationship where two expressions are not equal, but rather one is greater than, less than, greater than or equal to, or less than or equal to the other.  Unlike equations, inequalities have infinitely many solutions.

We represent inequalities using symbols:

*   <  Less than
*   >  Greater than
*   ≤  Less than or equal to
*   ≥  Greater than or equal to

**Example:**  x + 2 > 5  represents a situation where the value of 'x' must be greater than 3.

Let's consider the function f(x) = x + 2.  If we were to solve the inequality x + 2 > 5, we would isolate *x* to find x > 3. This *x* value is a part of the **domain** of f(x).

### 2. Solving Linear Inequalities

Solving linear inequalities involves isolating the variable using algebraic manipulations similar to solving equations. *However, when multiplying or dividing an inequality by a negative number, we must reverse the inequality sign.*

**Example 1:** Solve 3x - 2 < 7

1.  Add 2 to both sides:  3x < 9
2.  Divide both sides by 3:  x < 3

This means all values of *x* that are less than 3 are solutions to the inequality.  Graphically, this is represented by an open circle at 3 and a bracket extending to the left.

**Example 2:** Solve -2x + 5 ≥ 1

1.  Subtract 5 from both sides: -2x ≥ -4
2.  Divide both sides by -2 (and *reverse the inequality sign*): x ≤ 2

Again, all values of *x* that are less than or equal to 2 are solutions.  The graph would show a closed circle at 2 and a bracket extending to the left.

### 3. Graphical Representation of Inequalities

> **Note:** The graph of an inequality is not a single line like the graph of an equation. It’s a **region** on the number plane that satisfies the inequality.

The solution set of an inequality is represented graphically as a shaded region on the number plane. *Key features of the graph include open or closed circles at the boundary points and the direction of the shading.*

*   Open circle indicates the endpoint is *not* included in the solution.
*   Closed circle indicates the endpoint *is* included in the solution.

**Example:**  Consider the inequality x > 2.  The graph would be the region to the right of the vertical line at x = 2, with an open circle at x = 2.

## 4. Inequalities and Intervals

Inequalities can be expressed in **interval notation**. This provides a concise way to represent the solution set.

*   (a, b) represents all numbers between *a* and *b*, excluding *a* and *b* themselves (open interval).
*   [a, b] represents all numbers between *a* and *b*, including *a* and *b* (closed interval).
*   (a, ∞) represents all numbers greater than *a*, excluding *a* itself.
*   [a, ∞) represents all numbers greater than or equal to *a*.
*   (-∞, b] represents all numbers less than or equal to *b*, excluding *b* itself.
*   (-∞, b) represents all numbers less than *b*, excluding *b* itself.

**Example:** The solution to x < 3 is represented by the interval (-∞, 3).  The solution to x ≥ 1 is represented by the interval [1, ∞).

### 5. Inequalities with Composite Functions

Solving inequalities involving composite functions (like f(g(x))) requires careful application of the **chain rule** for inequalities. *You must solve the inner inequality first, then substitute the solution back into the outer inequality.*

**Example:** Solve f(x) = (x - 1)² + 3 > 0

1.  Isolate the squared term: (x - 1)² > -3
2.  Since the square of any real number is always non-negative, (x - 1)² is always greater than -3.  Therefore, this inequality is *always true* for all real values of *x*. The solution set is all real numbers.  The graph of f(x) is a parabola opening upwards, and the entire parabola is above the x-axis.

### 6.  Exam Practice - Application

> **Exam Tip:** IB exams often present word problems requiring you to translate real-world scenarios into mathematical inequalities. Pay close attention to the context and identify the variables and constraints.

**Problem:** A rectangle has a length that is 3 more than twice its width. The area of the rectangle is at most 100 cm².  Write an inequality to represent this situation and solve for the width.

1.  Let *w* represent the width of the rectangle.
2.  The length is 2*w* + 3.
3.  The area is length * width = (2*w* + 3) * *w* = 2*w*² + 3*w*
4.  The inequality: 2*w*² + 3*w* ≤ 100
5.  Rearrange: 2*w*² + 3*w* - 100 ≤ 0
6.  Solve the quadratic equation 2*w*² + 3*w* - 100 = 0 using the quadratic formula:  $w = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = \frac{-3 \pm \sqrt{3^2 - 4(2)(-100)}}{2(2)} = \frac{-3 \pm \sqrt{809}}{4}$
7.  Therefore,  $w \approx \frac{-3 \pm 28.44}{4}$. This gives us $w \approx 6.36$ or $w \approx -8.86$.
8.  Since the width cannot be negative, the solution is $w ≥ 6.36$ cm (approximately).

We've learned how to solve linear inequalities, express solutions in **interval notation**, and apply these concepts to composite functions.
