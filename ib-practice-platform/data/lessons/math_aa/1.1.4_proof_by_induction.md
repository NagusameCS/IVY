## Lesson: Proof by Induction

## Overview

This lesson introduces students to the method of mathematical induction, a powerful technique for proving statements about natural numbers. We will explore the core principles of the method, including the base case, inductive step, and how to construct a rigorous proof.

## Key Concepts

*   **Mathematical Induction:** A method of proving statements about all natural numbers.
*   **Base Case:** Establishing the truth of the statement for the smallest natural number (usually n=1).
*   **Inductive Step:** Assuming the statement is true for an arbitrary natural number *k*, and then proving that it must also be true for *k+1*.
*   **Principle of Mathematical Induction:** If a base case is established and the inductive step is proven, then the statement is true for all natural numbers.

### 1. Introduction to Mathematical Induction

> **Important:** Mathematical induction is a cornerstone of proof techniques in discrete mathematics, frequently appearing in the IB Mathematics Analysis and Approaches (AA) syllabus. It provides a systematic approach to demonstrating the validity of statements involving natural numbers.

Mathematical induction is particularly useful when dealing with formulas or sequences that involve *n*, where *n* represents a natural number (1, 2, 3, ...). It’s often used to prove statements about sums, sequences, or recursive algorithms. The core idea is to build a chain of reasoning: if the first case is true, and assuming the previous cases are true, then the next case *must* also be true.

**Example:** Consider the statement: “The sum of the first *n* natural numbers is given by 1 + 2 + 3 + ... + *n* = *n*( *n* + 1) / 2.” Let’s explore how induction can be used to prove this.

### 2. The Three Parts of Proof by Induction

A proof by induction consists of three distinct parts:

1.  **Base Case:** We prove that the statement is true for the smallest natural number, typically *n* = 1.
2.  **Inductive Hypothesis:** We *assume* that the statement is true for some arbitrary natural number *k*.  This is a crucial step – we are not *proving* it is true, merely *assuming* it is true for the sake of the argument.
3.  **Inductive Step:** We use the inductive hypothesis to prove that the statement is also true for *k* + 1.

> **Note:**  The inductive hypothesis is *not* a proven fact. It’s an assumption we make to simplify the proof.

### 3. Worked Example 1: Proving the Sum of First *n* Natural Numbers

Let's use this method to prove the formula 1 + 2 + 3 + ... + *n* = *n*( *n* + 1) / 2 using mathematical induction.

**Step 1: Base Case (n = 1)**

We need to show that the formula holds for *n* = 1.

Left-hand side (LHS): 1
Right-hand side (RHS): 1 * (1 + 1) / 2 = 1

Since LHS = RHS, the base case is established.

**Step 2: Inductive Hypothesis**

Assume that the formula holds for some natural number *k*. That is, we assume:

1 + 2 + 3 + ... + *k* = *k*( *k* + 1) / 2

**Step 3: Inductive Step**

We need to prove that the formula holds for *k* + 1.  That is, we need to show:

1 + 2 + 3 + ... + (*k* + 1) = (*k* + 1)( *k* + 2) / 2

Starting with the LHS:

1 + 2 + 3 + ... + (*k* + 1) = (1 + 2 + 3 + ... + *k*) + (*k* + 1)

Using the inductive hypothesis (1 + 2 + 3 + ... + *k* = *k*( *k* + 1) / 2), we get:

=  (*k*( *k* + 1) / 2) + (*k* + 1)
=  (*k*( *k* + 1) + 2(*k* + 1)) / 2
=  (*k*( *k* + 1) + 2*k* + 2) / 2
=  (*k*^{2} + *k* + 2*k* + 2) / 2
=  (*k*^{2} + 3*k* + 2) / 2
=  ((*k* + 1)(*k* + 2)) / 2

This is equal to the RHS. Therefore, the formula holds for *k* + 1.

> **Exam Tip:**  Always clearly state the inductive hypothesis.  It's a common source of errors if not explicitly defined.

### 4. Worked Example 2: Another Summation Proof

Let’s prove the formula for the sum of an arithmetic series:  S = *n*/2 [2*a* + (*n* - 1)*d], where *a* is the first term, *d* is the common difference, and *n* is the number of terms.

**Base Case (n = 1):** S = 1/2 [2*a* + (1 - 1)*d] = *a*.  This is true since the sum of one term is simply the term itself.

**Inductive Hypothesis:** Assume S = *n*/2 [2*a* + (*k* - 1)*d] holds for some integer *k*.

**Inductive Step:** We want to show S = *k* + 1/2 [2*a* + (( *k* + 1) - 1)*d] = *k* + 1.

S = *n*/2 [2*a* + (*n* - 1)*d]
= (*k* + 1)/2 [2*a* + (( *k* + 1) - 1)*d]
= (*k* + 1)/2 [2*a* + *k*d]
= (*k* + 1)/2 * 2*a* + (*k* + 1)/2 * *k*d
= (*k* + 1) *a* + (*k* + 1)/2 * *k*d

This approach demonstrates a slightly more complex proof, requiring careful manipulation of the formula.

### 5.  SL/HL Extension: Stronger Induction

> **Note:** At the HL level, 'stronger induction' (also known as 'complete induction') is often introduced. This requires *two* inductive steps:  first, to prove the statement for *n* = 1, and second, to show that if the statement is true for all *k* ≤ *n*, then it must also be true for *n* + 1.

### 6. Common Misconceptions

*   **Confusing the inductive hypothesis with a proven fact.** The inductive hypothesis is an *assumption* used to simplify the proof; it does not need to be proven true.
*   **Incorrectly applying the inductive step.** The inductive step must use the inductive hypothesis to show that the statement holds for the next number in the sequence.

## Summary

Proof by induction is a powerful technique for establishing the truth of statements about natural numbers. It relies on establishing a **base case** and then demonstrating that if the statement holds for an arbitrary *k*, it must also hold for *k* + 1.  This method is frequently used in mathematics, particularly for proving formulas involving sums and sequences. Mastering this technique is crucial for tackling problems within the IB Mathematics Analysis and Approaches curriculum (A.1.4).
