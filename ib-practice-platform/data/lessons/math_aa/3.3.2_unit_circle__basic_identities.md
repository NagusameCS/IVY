## Unit Circle & Basic Identities

## Overview

This lesson focuses on understanding the unit circle, its relationship to trigonometric functions, and utilizing fundamental trigonometric identities. Students will learn to accurately locate angles and their corresponding trigonometric values, and apply identities to solve problems.

## Key Concepts

*   Understanding the unit circle and its relationship to angles.
*   Determining the sine, cosine, and tangent of angles in any quadrant.
*   Applying **Pythagorean Identities** and **Reciprocal Identities** to solve equations and simplify expressions.
*   Using **co-functions** to relate sine and cosine values.

### Subtopic 1: The Unit Circle

> **Important:** The unit circle is a circle with a radius of 1 centered at the origin of a coordinate plane. It provides a visual representation of trigonometric functions.

The unit circle is fundamental to understanding trigonometric functions. The angle (θ) is measured in *radians* from the positive x-axis.  The coordinates of any point on the unit circle can be expressed using trigonometric functions. Specifically:

*   x-coordinate = cos(θ)
*   y-coordinate = sin(θ)

For a right-angled triangle in the first quadrant, with an angle θ, the sides are related by:

*   sin(θ) = Opposite / Hypotenuse
*   cos(θ) = Adjacent / Hypotenuse
*   tan(θ) = Opposite / Adjacent

These relationships translate directly to the unit circle.  For example, at an angle of π/2 (90°), cos(π/2) = 0 and sin(π/2) = 1.

**Example 1:**
Consider a right triangle with an angle θ = 30° in the first quadrant.  Using the unit circle, find:
    *   cos(30°)
    *   sin(30°)

**Solution:**
    *   cos(30°) = Adjacent / Hypotenuse = √3 / 2  (Looking at the unit circle, the x-coordinate at 30° is √3 / 2)
    *   sin(30°) = Opposite / Hypotenuse = 1 / 2 (Looking at the unit circle, the y-coordinate at 30° is 1 / 2)

### Subtopic 2: Trigonometric Values in All Quadrants

> **Note:** The sine and cosine functions are *odd functions*, meaning sin(-θ) = -sin(θ) and cos(-θ) = cos(θ). This allows us to easily find values in all four quadrants.

Trigonometric functions have the same *magnitude* but opposite signs in different quadrants.

*   **Quadrant I:** Sine and Cosine are positive.
*   **Quadrant II:** Sine is positive, Cosine is negative.
*   **Quadrant III:** Sine is negative, Cosine is negative.
*   **Quadrant IV:** Sine is negative, Cosine is positive.

**Example 2:**
Find the exact value of:
    *   sin(-45°)
    *   cos(135°)

**Solution:**
    *   sin(-45°) = -sin(45°) = -√2 / 2
    *   cos(135°) = cos(180° - 45°) = -cos(45°) = -√2 / 2 (Using the property cos(-θ) = cos(θ))

### Subtopic 3: Pythagorean Identities

> **Exam Tip:** Mastering these identities is crucial for solving trigonometric equations and simplifying expressions.

The Pythagorean identities are fundamental relationships between sine, cosine, and tangent:

*   **1. sin²(θ) + cos²(θ) = 1**
*   **2. 1 + tan²(θ) = sec²(θ)**
*   **3. 1 + cot²(θ) = csc²(θ)**

Where:

*   sec(θ) = 1 / cos(θ)
*   csc(θ) = 1 / sin(θ)
*   cot(θ) = 1 / tan(θ)

**Example 3:**
Given that sin(θ) = 3/5 and θ is in the first quadrant, find cos(θ) and tan(θ).

**Solution:**
    *   Using sin²(θ) + cos²(θ) = 1:
        (3/5)² + cos²(θ) = 1
        9/25 + cos²(θ) = 1
        cos²(θ) = 1 - 9/25 = 16/25
        cos(θ) = 4/5  (Since θ is in the first quadrant, cos(θ) is positive)
    *   tan(θ) = sin(θ) / cos(θ) = (3/5) / (4/5) = 3/4

### Subtopic 4: Reciprocal and Co-Function Identities

> **Note:** These identities are particularly useful for relating trigonometric functions with different arguments.

*   **Reciprocal Identities:**
    *   csc(θ) = 1 / sin(θ)
    *   sec(θ) = 1 / cos(θ)
    *   cot(θ) = 1 / tan(θ)

*   **Co-Function Identities:**
    *   sin(θ) = cos(90° - θ)
    *   cos(θ) = sin(90° - θ)
    *   tan(θ) = cot(90° - θ)
    *   cot(θ) = tan(90° - θ)

**Example 4:**
Find the value of cos(90° - θ).

**Solution:**
    *   Using the co-function identity: cos(90° - θ) = sin(θ)

### Subtopic 5: Solving Trigonometric Equations using Identities

> **Exam Tip:**  Start by applying relevant identities to simplify the equation.

Let’s use the Pythagorean identity to solve a trigonometric equation.

**Example 5:**
Solve the equation sin²(θ) + cos²(θ) = 1 for θ.

**Solution:**
    *   This equation is already a fundamental trigonometric identity.  Therefore, it’s always true for any value of θ.  This illustrates how identities can be used to confirm known relationships.

### Subtopic 6: Applying Identities to Simplify Expressions

> **Important:** Practice simplifying trigonometric expressions is key to solidifying your understanding.

**Example 6:**
Simplify the expression:  (1 - tan²(θ)) / (1 + tan²(θ))

**Solution:**
    *   Using the Pythagorean identity: 1 + tan²(θ) = sec²(θ)
    *   The expression becomes: (1 - tan²(θ)) / sec²(θ)
    *   Now, rewrite sec²(θ) as 1 / cos²(θ):  (1 - tan²(θ)) * cos²(θ)
    *   Since tan(θ) = sin(θ) / cos(θ), we have: (1 - (sin²(θ) / cos²(θ))) * cos²(θ)
    *   = (cos²(θ) - sin²(θ)) / cos²(θ) * cos²(θ)
    *   Using the Pythagorean identity sin²(θ) + cos²(θ) = 1, we know cos²(θ) - sin²(θ) = cos(2θ)
    *   Therefore, the simplified expression is: cos(2θ)
