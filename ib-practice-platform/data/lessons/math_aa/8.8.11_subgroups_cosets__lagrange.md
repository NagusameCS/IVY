## Overview

This is an IB Math course about sets and relations in the context of Lagrange's theorem. It covers topics such as:

1. **Sets**: A set can be thought of as a collection of distinct objects or elements, like our 2-dimensional Cartesian plane.

2. **Representation**: An object (in this case, an integer) can be represented by two sets (the "set" and the "complement"). For example, the integers {1,2} represent the set of all positive numbers, while the complement of that set is {-1, 3}.

3. **Lagrange's Theorem**: Lagrange showed that there are no proper subspaces containing a given subset, except possibly as the limit or limit point of it.

**Teaching Strategy:**

1. **Real Number System**: Students work with real numbers and develop their understanding of set operations, including the concept of sets in relation to subsets.

2. **Representation**: Students use sets as collections of elements (the "representation"), which form a basis for representing any other object or element.

3. **Lagrange's Theorem:** After learning about subsets, they can apply Lagrange's theorem and see how it relates sets to subspaces in general.

4. **Application:** The lesson provides an example where the complement of an integer is equal to -1 (i.e., a set that contains no integers), which aligns with Lagrange's theorem.
