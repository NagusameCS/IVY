## Set Operations & Laws

## Overview

This lesson covers fundamental **set operations** and the **laws of set algebra** used throughout mathematics.

## Key Concepts

- Union, intersection, complement, and difference
- De Morgan's laws
- Distributive, associative, and commutative laws
- Venn diagrams

### Basic Set Operations

Let $A$ and $B$ be subsets of a universal set $U$.

| Operation | Notation | Definition |
|---|---|---|
| Union | $A \cup B$ | $\{x : x \in A \text{ or } x \in B\}$ |
| Intersection | $A \cap B$ | $\{x : x \in A \text{ and } x \in B\}$ |
| Complement | $A'$ | $\{x : x \in U \text{ and } x \notin A\}$ |
| Difference | $A \setminus B$ | $\{x : x \in A \text{ and } x \notin B\}$ |
| Symmetric Difference | $A \triangle B$ | $(A \setminus B) \cup (B \setminus A)$ |

### Laws of Set Algebra

> **Important:** These laws are analogous to laws of logic and can be proved using element arguments or Venn diagrams.

**Commutative Laws:**
- $A \cup B = B \cup A$
- $A \cap B = B \cap A$

**Associative Laws:**
- $(A \cup B) \cup C = A \cup (B \cup C)$
- $(A \cap B) \cap C = A \cap (B \cap C)$

**Distributive Laws:**
- $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

**De Morgan's Laws:**
- $(A \cup B)' = A' \cap B'$
- $(A \cap B)' = A' \cup B'$

**Identity Laws:**
- $A \cup \emptyset = A$
- $A \cap U = A$

**Example:** Prove that $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$.

Let $x \in A \cap (B \cup C)$. Then $x \in A$ and $x \in B \cup C$, so $x \in B$ or $x \in C$.
- If $x \in B$: then $x \in A \cap B$, so $x \in (A \cap B) \cup (A \cap C)$
- If $x \in C$: then $x \in A \cap C$, so $x \in (A \cap B) \cup (A \cap C)$

The reverse inclusion follows similarly.

## Summary

- Set operations (union, intersection, complement, difference) are the building blocks of set theory
- De Morgan's laws relate complements of unions and intersections
- Distributive laws allow expanding or factoring set expressions
